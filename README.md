# Civic Grievance Portal

A comprehensive complaint management system for citizens to file grievances and government departments to track, assign, and resolve issues with auto-escalation.

Spring Boot | React | PostgreSQL

---

## Features

### For Citizens
- Register/Login with Aadhaar & OTP
- File complaints with photo/video evidence
- Auto-fetch location using GPS
- Track complaint status in real-time
- View all your complaints

### For Department Officials
- Department-wise dashboard with statistics
- Approve/Reject complaints
- Assign complaints to field solvers
- Track department performance
- View pending and completed complaints

### For Field Solvers
- View assigned tasks
- Start work on complaints
- Upload completion photos with comments
- Mark tasks as completed
- Track personal performance stats

### Smart Features
- Auto-escalation after deadline (1-minute demo mode)
- Real-time status updates
- Live countdown timers
- Fully responsive design
- Role-based access control

---

## Tech Stack

### Backend
- Framework: Spring Boot 3.1.5
- Database: PostgreSQL
- ORM: Spring Data JPA (Hibernate)
- Security: JWT Authentication
- Build Tool: Maven
- Java Version: 17+

### Frontend
- Framework: React
- Styling: Tailwind CSS
- Animations: Framer Motion
- HTTP Client: Axios
- Routing: React Router v6
